User Guide: Location Based Quiz - Question Setting App

Displaying Questions Already Set

• Firstly, ensure the httpServer.js is running. This can be found in the server repository (https://github.com/nikscrits/server)

• Press the ‘Show Points’ button at the bottom of the page.

• The questions will appear as orange markers on the map and the map will automatically pan to give a view showing all of the question markers (see image below).

• Click on any marker in order to see its Point name and Question information. Additionally, the zoom function on the map can be used to view points in more detail.

Submitting a New Question

• To create new questions for the user to answer the form at the bottom of the page must be filled out (see image below). Note: ensure you fill in every text box or the question will not upload and an alert will show. This involves filling in the following information:

1. A question - this can include letters, numbers and punctuation. It must be under 200 characters in length.

2. Four potential answers to the question – this should include the correct answer as one of the options. This can include letters, numbers and punctuation. It must be under 50 characters in length.

3. The question location longitude and latitude – this can be entered manually or by clicking on the location on the map.

4. The correct answer – the relevant radio button should be selected indicating which answer is correct.

• If you wish to clear the form to start entering the details again, press the ‘Reset Form’ button.

• Click the ‘Upload Question’ button to send the question to the database. Two alerts will then be presented. The first alert shows the data that will be sent to the databse and the second shows that the data has been successfully added.

• In order to view this new question point, refresh the webpage and click the ‘Show Points’ button. The point that has been added should now appear as a marker (see image below).
